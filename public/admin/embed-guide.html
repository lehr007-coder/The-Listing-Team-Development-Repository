GoHighLevel Embed Guide

How to use your images in GHL funnels, websites, and widgets

Method 1: Direct Image URL (Simplest)

Use your image URLs directly in any GHL image element, background image, or HTML block.

1 Copy the image URL from Admin

Go to the Admin Portal, find your image, and click "Copy URL".

2 Paste in GHL

In the GHL funnel builder:

  • Image Element: Paste the URL in the "Image URL" field
  • Background Image: Paste in section/row background image URL
  • Custom HTML: Use an img tag:
<img src="https://YOUR-SERVER/images/your-image-slug" alt="Your alt text" style="width: 100%; max-width: 600px; height: auto;" >

Method 2: Single Image Widget

Embed a single image that automatically pulls its alt text and description from the server.

1 Add a Custom HTML/JS element in your GHL funnel

2 Paste this code:

<!-- TLT Single Image Widget --> <div class="tlt-image" data-slug="YOUR-IMAGE-SLUG" data-max-width="600px" data-rounded="true"> </div> <script src="https://YOUR-SERVER/embed/widget.js" data-server="https://YOUR-SERVER"> </script>

Options:

  • data-slug - The image slug (required)
  • data-max-width - Max width (default: 100%)
  • data-rounded - Rounded corners: true/false

Method 3: Image Gallery Widget

Display a responsive grid of images from your server.

1 Add a Custom HTML/JS element in your GHL funnel

2 Paste this code:

<!-- TLT Image Gallery Widget --> <div class="tlt-gallery" data-columns="3" data-limit="6" data-captions="true" data-rounded="true" data-gap="16px"> </div> <script src="https://YOUR-SERVER/embed/widget.js" data-server="https://YOUR-SERVER"> </script>

Options:

  • data-columns - Grid columns (default: 3)
  • data-limit - Max images to show (default: 12)
  • data-search - Filter images by keyword
  • data-captions - Show title/description: true/false
  • data-rounded - Rounded corners: true/false
  • data-gap - Spacing between images (default: 16px)

Step-by-Step in GoHighLevel

For Funnels:

  • 1 Open your funnel in the GHL builder
  • 2 Add a new element → select "Custom JS/HTML"
  • 3 Paste the widget code from above
  • 4 Replace YOUR-SERVER with your image server domain
  • 5 Replace YOUR-IMAGE-SLUG with the actual slug
  • 6 Save and preview

For Websites:

  • 1 Open your website in the GHL builder
  • 2 Add a "Code" element to any section
  • 3 Switch to "Custom Code" and paste the widget code
  • 4 Update the server URL and image slug
  • 5 Save and publish
Tip: You can add multiple <div class="tlt-image"> elements in the same HTML block, but only include the <script> tag once at the bottom.

Custom Styling in GHL

Add custom CSS in the same HTML element to override default styles:

<style> .tlt-image img { border-radius: 12px; box-shadow: 0 4px 20px rgba(0,0,0,0.15); transition: transform 0.3s; } .tlt-image img:hover { transform: scale(1.02); } .tlt-gallery img { border-radius: 8px; } </style>
